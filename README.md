# API to Excel Automation (Python)

A simple Python automation that fetches data from a REST API and exports it into a clean Excel report.

---

## What It Does
- Fetches data from an API
- Processes JSON data safely
- Exports a formatted Excel file

---

## Technologies
- Python
- requests
- pandas
- openpyxl

---

## How to Run
`bash
pip install -r requirements.txt
python src/main.py


---

## Output

The Excel file is generated inside the output/ folder.

A sample output is available in examples/.
